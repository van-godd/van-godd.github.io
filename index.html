Resources

Here are some good websites and useful resources I use regularly:


- development -

-> vercel [link] A really good web-hosting platform that offers many features for free! I use it in almost all my projects.

-> MDN-Docs [link] My go-to platform if I have any problems with HTML/CSS/JS, or for exploring all the posibilities!

-> fontshare [link] Fonts for your website, similar to google-fonts. The library is a bit smaller, but with the same functionality and without trackers!

- graphics -

-> photopea [link] A good browser-based image editor for quick edits!

-> postimages [link] Best service for hosting your images, without compressing them!

-> caesium-ic [link] Desktop software to optimize your images to make your page load faster.

-> raw therapee [link] A free Lightroom alternative for high quality color grading.

- inspiration for web-design -

-> awwwards [link] Find websites with unique and outstanding designs.

-> brutalistwebsites [link] A list of websites with extreme designs.

-> three js [link] A library for advanced 3D-websites, also features a lot of good sites.
